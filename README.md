## Usage

1. Include jQuery:

  ```html
  <script src="http://ajax.googleapis.com/ajax/libs/jquery/2.0.0/jquery.min.js"></script>
  ```

2. Include plugin's code:

  ```html
  <script src="dist/jquery.quadgram.min.js"></script>
  ```

3. Call the plugin:

  ```javascript
$('#table').jq_quadgram({
	values: [1, 2, 3, 4]
});
  ```

See [demo page](demo/index.html) for more information.
